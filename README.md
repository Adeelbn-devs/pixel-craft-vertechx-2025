‚ú® Features
This is a multi-page, fully responsive website built with a modern UI/UX approach. All dynamic functionality is implemented using Vanilla JavaScript, with no external libraries (like jQuery or Bootstrap).

Multi-Page Structure:

Home Page (index.html): Features the main hero, about, stats, gallery, and theme sections.

About Page (about-more.html): Provides detailed information about the college and the fest.

Theme Page (pixelate-more.html): Explains the "Pixelate" theme concept.

Modern UI & Design:

Dark/Light Mode: A smooth theme toggle that saves the user's preference in localStorage.

Fully Responsive: A mobile-first design that ensures a seamless experience on all screen sizes.

Modern Gradient Palette: Utilizes CSS Custom Properties for a clean and beautiful blue gradient theme.

Dynamic JavaScript Features:

Auto-Typing Effect: An engaging auto-typing animation in the hero section.

Scroll-Triggered Animations:

Scroll Reveal: Elements gracefully fade in as the user scrolls.

Sticky Navbar Effect: The navigation bar shrinks and changes background on scroll.

Parallax Background: A subtle parallax effect on the hero section image.

Interactive UI Elements:

Count-Up Stats: Numbers in the stats section animate from 0 to their target value.

Active Nav Highlighting: Navigation links are automatically highlighted based on the current scroll position.

Button Ripple Effect: A clean, material-style ripple animation on button clicks.

Performance & Accessibility:

Includes performance optimizations like lazy loading (placeholder) and respects the prefers-reduced-motion media query.

CSS3: Advanced CSS including Flexbox, Grid, Custom Properties, Gradients, and Keyframe Animations.
